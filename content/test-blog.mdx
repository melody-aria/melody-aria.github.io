# 新书小白如何通过github搭建个人博客

## 博客介绍

这是一个基于 GitHub Pages 的个人博客，采用了 Next.js 框架和 MDX 格式。博客的内容主要是关于宫崎骏的电影作品，包括电影、书籍、游戏等。

### 博客的搭建过程

1. **创建 GitHub 仓库**：首先，在 GitHub 上创建一个新的仓库，仓库名称为 `your-username.github.io`，其中 `your-username` 是您的 GitHub 用户名。

2. **克隆仓库到本地**：使用 Git 克隆您的仓库到本地计算机。

3. **安装依赖**：在项目根目录下，使用 npm 或 yarn 安装项目依赖。

4. **配置 Next.js**：根据 Next.js 文档，配置项目的路由、页面等。

5. **编写博客文章**：在 `content` 目录下创建新的 MDX 文件，编写您的博客文章。

6. **部署到 GitHub Pages**：使用 GitHub Actions 自动部署您的博客到 GitHub Pages。

### 博客的维护与更新

1. **添加新文章**：在 `content` 目录下创建新的 MDX 文件，编写新的博客文章。

2. **更新配置**：如果您添加了新的文章或页面，需要更新 `next.config.js` 中的路由配置。

3. **提交更改**：使用 Git 提交您的更改，并推送到 GitHub 仓库。

4. **自动部署**：GitHub Actions 会自动检测到您的更改，并触发自动部署。

### 博客的自定义与扩展

1. **自定义主题**：您可以根据自己的喜好，自定义博客的主题样式。

2. **添加插件**：Next.js 支持添加插件，您可以根据需要添加插件，如评论插件、分析插件等。

3. **部署到自定义域名**：如果您有自己的域名，您可以将博客部署到自定义域名下。

### 博客的访问与分享

1. **访问博客**：您可以通过 `https://your-username.github.io` 访问您的博客。

2. **分享博客**：您可以将博客分享到社交媒体上，如 Twitter、Facebook、LinkedIn 等。
