# 前言
    每个人大学毕业后到陌生的城市工作，第一件事情就是找到一个容身之处。
    那么巧了，中介和房东最喜欢的就是刚毕业不谙世事的大学生。看着他们清澈愚蠢的眼神，嘴都列脖子后了。
    我也是刚毕业，踩了很多坑，最终也是租到了一个满意的房子。
# 租房补贴
    刚毕业的应届毕业生，一定一定要去查一下当地有没有租房补贴。
    我所在的城市是有租房补贴的，一个月有800块钱。不要小看这800块钱，这是一个很大的帮助,尤其是在刚毕业的阶段。
    租房补贴首先可以帮助你缓解经济上的压力，其次一般租一个比较好的房子的话，在我所在的城市一般是在1000-1500一个人就可以租一个很好的房子了。
    我现在是租的房子是1200元一个月，不包括水电，水电都是民用的。那实际上我需要支付的房租就是400块。
## 如何申请租房补贴
    租房补贴必须是需要房屋租赁备案才能申请。
    --------------------------------
    只有小区房有房产证才能进行房屋租赁备案的申请。
    流程并不算麻烦，找到自己心仪的房子之后，和房东沟通之后拿到租房合同、房产证和房东的身份证复印件、消防设备套装等资料，如果还有额外资料，需要你去所在小区的社区询问。
    线下的话，需要你和房东都去社区进行办理，还需要警卫室进行签字审核。
    线上则需要在当地的公众号查看对应文章进行申请。
    我是直接在线上申请的，填写了相应的资料，但是当时我等了很久都没有通过，最后我还是去线下去社区去询问，才知道人家压根就没有管过线上的申请，我说过后他们才知道。
    去线下还是比较稳妥的
    --------------------------------
    注意：以上步骤主要是为了拿到房屋租赁备案
    拿到房屋租赁备案后，需要去当地的办事大厅进行申请。 需要提供学信网的学业证明。
## 发放
    我所在的城市是一个季度发放一次，具体的发放时间并不确定。
    我在10月份申请通过了，但是必须要等26年第一季度才能发放。
# 租房注意事项
    1.如果是中介推荐给你，一定要去中介的平台查看一下，中介是否有什么问题。并且中介费一定是房租的一般，要全额房租或者更高，就直接pass吧。
    2.房租一定要线下多看多对比，许多app上可以全景看房，但是谁知道这些图片是多久前的了。
    3.如果这个房子看着装修很新，里面设施也很新，但是价格很低，那很有可能就是串串房，或者有其他问题，一定要仔细分辨。
    4.如果不是非常非常缺钱，一定一定不要合租。很难想象你会遇到什么神人，如果合租舍友是正常人还好，不正常的话那就非常非常的难受。本来上班就很累了，还要应付这些神仙。
    5.可以去小红书直接去发布自己的租房需求，这样可以直接联系到很多人，也可以直接联系到房东。当然，也可以直接去搜索房东直租的房屋信息。
       我就是在小红书上找到房东直租的房屋信息，然后直接联系到了房东。房东也非常好，很负责。房子也令我满意。
    6.如果没有找到合适的房子，可以去酒店租一个月租房，价格应该和当地的房租价格是差不多的，有些酒店不用支付水费而且还可以免费打扫卫生。也是很舒服的。
# 总结
    本文是对我租房经历的一个记录，也是我在租房过程中遇到的一些问题和解决方法。
    租房不能着急，找到适合的、喜欢的房子是非常重要的，一个好的房子会影响到你生活的方方面面，相反租到一个不满意的房子，下班回到家都会很难受。
