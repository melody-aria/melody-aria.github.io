# v-for="item in 7"
    我们可以使用v-for指令来循环渲染元素。
    语法：
    ```
    <div v-for="item in 7">
      {{ item }}
    </div>
    ```
    渲染结果：
    ```
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
    <div>6</div>
    <div>7</div>
    ```
