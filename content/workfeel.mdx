# 迷茫

回看大学四年，其实自己没有真正做过什么对能力提升有帮助的事。  
大一大二基本下课就回宿舍打游戏，我并不觉得有问题，只要保证不挂科就行；想拿奖学金，临时抱佛脚也够用了。  

真正让我担心的是：我到底要做什么？做什么才对自己有帮助？该选哪条路？  

爸妈想让我考公、考研；哥哥劝我学一门技术，直接当程序员。  
我全然不知该干什么，也不知道什么才是“最适合”我的选择。  

家庭条件特殊，当时觉得考公、考研都不是好选项：  
- 需要投入大量时间与金钱  
- 一旦落榜，全部打水漂  
- 以我当时的状态，上岸够呛  

即便侥幸上岸，考研还要再读两三年，又是一笔巨大支出，对家庭是沉重负担。  
于是，我先把“考研”划掉了，却仍找不到方向。

# 小转折

大三那年，学校合作了一个培训班：Java 与嵌入式。  
和哥哥商量后，我决定学 Java。  

培训两三个月后，就业老师推荐了几家企业：实施、测试、运维、开发。  
三分之二的同学去了跟 Java 毫无关系的实施或运维，我不想去——学费不就白交了？  
只能硬着头皮找开发岗。  

运气不错，和两个同学一起进了国内排名靠前的 ERP 公司。  
可笑的是，从入职到离职，我连公司本部都没去过，直接被派到客户现场驻场。  
领导和师傅人很好，工作不算艰难，但能力提升有限：  
- 公司用自研框架，跟主流技术栈脱节  
- 自己眼界也浅，没主动钻研，代码能力停滞不前  

干了半年，公司组织架构调整，部门领导被降职，同学和师傅相继离职。  
我也不想继续，拿着实习攒下的“三瓜俩枣”，回校享受最后的大学时光。  

那时已隐约察觉：我似乎并不喜欢写代码。  
直到毕业，我都没再找新工作，只是一味逃避——害怕面试，害怕下一份工作。

# 毕业后

毕业后我住在北京哥哥家，一边复习后端知识，一边投简历。  

印象最深、也最难受的一件事：  
学校招劳务派遣岗，我很想去，辅导员甚至把推荐名额留给了我，票都差点买了。  
却因和哥哥意见不合，大吵一架，最终放了辅导员“鸽子”。  

至今想起仍愧疚：连面试都没去，浪费了辅导员的好意。  
如今对比现状，更觉难受。  

之后便是漫长的“投简历—面试”循环，最终入职一家互联网公司，做 Java 开发。

# 入职三个月以来的感受

入职后，我每天都很焦虑：  
1. 我发现自己真的不喜欢写代码  
2. 我仿佛一眼就能看到未来的生活  

每天都在想：  
- 不写代码，我还能干什么？  
- 试用期过不了怎么办？  

每次学习，知识都是左耳进右耳出，留不下分毫。  
不知这是学习能力退化，还是对厌恶之事的本能抗拒。  

有一段时间，我极度抑郁，极度消极。
