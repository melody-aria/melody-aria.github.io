# Markdown 语法速查表

这是一份完整的 Markdown 语法参考指南，帮助你快速掌握常用的 Markdown 语法。

## 📝 标题语法

使用 `#` 符号创建标题，支持 1-6 级标题：

```
# 一级标题
## 二级标题  
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果展示：**
# 一级标题示例
## 二级标题示例
### 三级标题示例

---

## ✍️ 文本格式

### 基础格式

```
*斜体文本* 或 _斜体文本_
**粗体文本** 或 __粗体文本__
***粗斜体文本*** 或 ___粗斜体文本___
~~删除线文本~~
```

**效果展示：**
- *斜体文本*
- **粗体文本**  
- ***粗斜体文本***
- ~~删除线文本~~

### 段落与换行

```
这是第一个段落。

这是第二个段落。

如果要在同一段落内换行，  
在行末添加两个空格然后回车。
```

---

## 📋 列表语法

### 无序列表

使用 `-`、`*` 或 `+` 创建无序列表：

```
- 项目一
- 项目二
  - 子项目 2.1
  - 子项目 2.2
- 项目三
```

**效果展示：**
- 项目一
- 项目二
  - 子项目 2.1
  - 子项目 2.2
- 项目三

### 有序列表

使用数字加点创建有序列表：

```
1. 第一项
2. 第二项
3. 第三项
   1. 子项目 3.1
   2. 子项目 3.2
```

**效果展示：**
1. 第一项
2. 第二项
3. 第三项
   1. 子项目 3.1
   2. 子项目 3.2

---

## 🔗 链接和图片

### 链接语法

```
[链接文本](https://example.com)
[带标题的链接](https://example.com "这是标题")
<https://example.com>
```

**效果展示：**
- [百度](https://baidu.com)
- [带标题的链接](https://baidu.com "百度首页")

### 图片语法

```
![替代文本](图片URL)
![替代文本](图片URL "图片标题")
```

## 💬 引用语法

使用 `>` 创建引用：

```
> 这是一级引用
> 
> > 这是二级引用
> > 
> > > 这是三级引用
```

**效果展示：**
> 这是一级引用
> 
> > 这是二级引用
> > 
> > > 这是三级引用

---

## 💻 代码语法

### 行内代码

使用反引号包围代码：

```
这是 `行内代码` 示例
```

**效果展示：**
这是 `行内代码` 示例

### 代码块

使用三个反引号创建代码块：

````
```javascript
function hello() {
    console.log("Hello, World!");
}
```
````

**效果展示：**
```javascript
function hello() {
    console.log("Hello, World!");
}
```

## 📊 表格语法

```
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
| 王五 | 28   | 广州 |
```

**效果展示：**
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
| 王五 | 28   | 广州 |

### 表格对齐

```
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   |   内容   |   内容 |
```

---

## ➖ 分隔线

使用三个或更多的 `-`、`*` 或 `_` 创建分隔线：

```
---
***
___
```

---

## ☑️ 任务列表

```
- [x] 已完成的任务
- [ ] 未完成的任务
- [x] 另一个已完成的任务
```

**效果展示：**
- [x] 已完成的任务
- [ ] 未完成的任务
- [x] 另一个已完成的任务

---

## 🔤 转义字符

使用反斜杠 `\` 转义特殊字符：

```
\# 这不是标题
\*这不是斜体\*
\**这不是粗体\**
\- 这不是列表项
```

**效果展示：**
\# 这不是标题  
\*这不是斜体\*  
\**这不是粗体\**  
\- 这不是列表项

---

## 📝 脚注（扩展语法）

```
这是一个脚注示例[^1]。

[^1]: 这里是脚注的内容。
```

---

## 📁 折叠内容（HTML 支持）

```html
<details>
<summary>点击展开详细内容</summary>

这里是折叠的内容，点击上方可以展开或收起。

可以包含任何 Markdown 语法：
- 列表项
- **粗体文本**
- `代码`

</details>
```

**效果展示：**
<details>
<summary>点击展开详细内容</summary>

这里是折叠的内容，点击上方可以展开或收起。

可以包含任何 Markdown 语法：
- 列表项
- **粗体文本**
- `代码`

</details>

## 🎯 实用技巧

### 1. 快速创建表格
可以使用在线表格生成器，然后复制 Markdown 代码。

### 2. 代码高亮
在代码块中指定语言可以获得语法高亮：
- `javascript`、`python`、`java`、`css`、`html` 等

### 3. 链接引用
对于重复使用的链接，可以使用引用式链接：

```
[Google][1]
[GitHub][2]

[1]: https://google.com
[2]: https://github.com
```

### 4. 图片尺寸控制（HTML）
```html
<img src="image.jpg" width="300" height="200" alt="描述">
```

## 📚 总结

掌握这些 Markdown 语法，你就能应付绝大多数的写作需求了。记住：

1. **标题**：用 `#` 号
2. **强调**：用 `*` 或 `_`
3. **列表**：用 `-` 或数字
4. **链接**：用 `[文本](URL)`
5. **代码**：用反引号 `` ` ``
6. **表格**：用 `|` 分隔

多练习，很快就能熟练掌握！🚀
