# div标签
```
<template>
  <div>
    <div>
      <div>1.1</div>
      <div>1.2</div>
    </div>
    <div>2</div>
  </div>
</template>

<style scoped>
</style>
```
div标签
  *div标签*：用于定义文档中的分区或者节的组合。
  *注意*：div标签是一个块级元素，会独占一行。
*div标签就类似一个房子用来装其他的元素*

# 调整高度
  *height*：用于设置元素的高度。
  *注意*：height属性的单位是px（像素）。
# margin
  *margin*：用于设置元素的外边距。
  *注意*：margin属性的单位是px（像素）。
  *margin-right*：用于设置元素的右边外边距。
  *margin-top*：用于设置元素的上边外边距。
  *margin-left*：用于设置元素的左边外边距。
  *margin-bottom*：用于设置元素的下边外边距。
# display
  *display*：用于设置元素的显示方式。
  *注意*：display属性的单位是px（像素）。
  ## inline-block：用于设置元素的显示方式为行内块元素。
  *inline*：用于设置元素的显示方式为行内元素。
  什么是行内块元素？
  *行内块元素*：行内块元素是指元素的显示方式为行内元素，但是可以设置宽度和高度。
  什么是行内元素？
  *行内元素*：行内元素是指元素的显示方式为行内元素，不能设置宽度和高度。
  
# span标签
  *span标签*：用于定义文档中的行内元素。
  *注意*：span标签是一个行内元素，不能设置宽度和高度。
# 切换背景颜色
  *background-color*：用于设置元素的背景颜色。
  *注意*：background-color属性的单位是px（像素）。
